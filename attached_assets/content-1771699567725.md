# 🚀 Iago Barreto — Full-Stack Developer & Product Builder

[Permalink: 🚀 Iago Barreto — Full-Stack Developer & Product Builder](https://github.com/iaguu#-iago-barreto--full-stack-developer--product-builder)

Sou um desenvolvedor focado em **criar soluções completas**, unindo **frontend, backend, mobile, APIs, automações**, experiência do usuário e visão de produto. Trabalho com desenvolvimento moderno usando **JavaScript/TypeScript**, com forte domínio em **React, React Native/Expo, Node.js, Electron, Vite, Express, SCSS**, além de integrações com **gateways de pagamento**, sistemas externos e infraestrutura leve.

Atuo construindo sistemas do zero, desde a **arquitetura**, **UI/UX**, **backend**, **banco de dados**, **deploy**, até **otimizações de produto e negócio**. Minha especialidade é transformar operações reais em **plataformas robustas, rápidas e eficientes**.

### **Anne & Tom – Sistema completo de pizzaria**

[Permalink: Anne & Tom – Sistema completo de pizzaria](https://github.com/iaguu#anne--tom--sistema-completo-de-pizzaria)

Uma suíte completa para gestão de uma pizzaria moderna:

- **Website React** com cardápio dinâmico, checkout e pedidos

- **Sistema Desktop Electron** com:
  - Caixa diário (auto-fechamento + PDF + impressão)
  - Dashboard de vendas
  - Gestão de pedidos com impressão silenciosa
  - Cadastro de clientes com busca automática
  - Estoque, produtos, categorias, configurações
- **API Node.js/Express** para comunicação Web ←→ Desktop

- **App Mobile (motoboy)** com:
  - Leitura de QR Code
  - Rastreamento em tempo real via GPS
  - Atualização de status de entrega

Tudo integrado em um **ecossistema único**, com sincronização, design profissional e fluxo contínuo de dados.
